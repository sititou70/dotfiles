# dotfiles

## setup

1. clone the repository to your `$HOME`
1. install dependencies
   - Ubuntu: `sudo apt install -y unzip curl wget vim tmux`
   - macOS: `brew install unzip curl wget vim tmux`
   - Ubuntu Server: `sudo apt install -y unzip curl wget vim`
1. install dotfiles: `./install.sh`

## uninstall

`./uninstall.sh`

# dotfiles

## setup

1. clone the repository to your `$HOME`
1. install dependencies
   - Ubuntu: `sudo apt install -y git unzip curl wget make zsh vim tmux eza xsel htop nvtop shfmt; snap install btop`
   - macOS: `brew install coreutils unzip curl wget make vim zsh tmux eza btop htop nvtop shfmt`
   - Ubuntu Server: `sudo apt install -y git unzip curl wget make zsh vim eza htop`
1. install dotfiles: `./install.sh`

## uninstall

`./uninstall.sh`
